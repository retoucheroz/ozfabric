Bu workflow, ozfabric projesini yeni bir ortamda hızlıca ayağa kaldırmak için kullanılır.

// turbo-all
1. Bağımlılıkları yükle ve sistemi kontrol et:
```bash
./setup.sh
```

2. Geliştirme sunucusunu başlat:
```bash
npm run dev
```

3. Kurulum tamamlandıktan sonra Antigravity'ye "Sistemi kontrol et" diyerek son durumu doğrulayabilirsiniz.
